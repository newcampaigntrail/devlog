The NCT Modmaker

2023-05-14

As a part of the domain shift, the modmaker has been moved to a new subdomain at modmaker.newcampaigntrail.com.

Since we didn't really announce the redux of the modmaker at any point, here is a quick list of the modmaker capabilities:

  • Preview function built into the NCT engine.
  • No requirement of working with PK relations - the maker deals with it all for you.
  • Portrait size requirement warnings to ensure mods have appropriately sized images.
  • Theme builder with colour pickers.
  • Clear no-code interface.

And here is a list of limitations:

  • For now: code 1 only. If you would like a code 2 modmaker, have a look at the fantastic Jet's TCT Mod Tool (DISCLAIMER: not affiliated with the NCT team).
  • The import function has not yet been added. I cannot stress enough that you should not rely on it. However, you can still, of course, export your code 1 for later use by yourself.

In the future, expect more updates for the modmaker.
